# deportes

Favorite Football Players – Vue 3 app that uses **API-Football** (api-sports.io) for player search and analysis. Full player info and **club(s)** are shown.

- **API:** [API-Football](https://www.api-football.com/documentation-v3) at `https://v3.football.api-sports.io`

### Environment

Create a `.env` file with your API-Football key (free at [api-football.com](https://www.api-football.com/)):

```env
VITE_FOOTBALL_API_KEY=your-key-here
```

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
